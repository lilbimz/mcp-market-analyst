# MCP Market Analyst

An MCP (Model Context Protocol) Server for querying real-time cryptocurrency data and global stock/forex market data.

## Features
- Binance public API integration for crypto candles.
- yfinance integration for stock/forex data.
- Built using FastAPI with SSE Transport.
